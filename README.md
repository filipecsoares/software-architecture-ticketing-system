## 📚 About

This repository documents my journey through the book "Microservices with Spring Boot 3 and Spring Cloud" by Magnus Larsson, but with a modern twist—I'm using the **latest versions** of Spring Boot, Spring Cloud, and all related technologies.

Each chapter is implemented as a distinct milestone, adapting the book's concepts to leverage the newest features while handling breaking changes along the way.
